# Peluqueria-Canina

PROYECTO PELUQUERIA CANINA

Proyecto de una aplicación realizado con Swing para una peluqueria canina. 
Permite realizar una CRUD de mascotas y dueños conectando la aplicación con una base de datos MySQL.
